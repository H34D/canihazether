### Can I Haz Ether?

Find Ether on known Ethereum Networks.


### Installation

```bash
npm install -g canihazether
```

### Usage

`canihazether [address]`

```bash
$ canihazether 0x742d35cc6634c0532925a3b844bc454e4438f44e

Searching ...
Address 0x742d35cc6634c0532925a3b844bc454e4438f44e haz:

1962424.438426623500000001 Ether on the Ethereum network (0x1).

searched on 45 networks from the atlas
```
